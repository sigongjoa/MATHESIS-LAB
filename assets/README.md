# Assets 폴더

이 폴더는 RAG 시스템에서 사용할 **원본 교육 문서**들을 보관하는 곳입니다.

## 📁 폴더 구조

```
assets/
├── curriculum/           # 국가 교육과정 문서
│   ├── 2022/            # 2022 개정 교육과정
│   │   ├── [별책 8] 수학과 교육과정.pdf
│   │   ├── [별책 9] 과학과 교육과정.pdf
│   │   └── ...
│   ├── 2015/            # 2015 개정 교육과정
│   │   └── ...
│   └── 2009/            # 2009 개정 교육과정
│       └── ...
│
├── schools/             # 개별 학교 운영 계획
│   ├── 협성고/
│   │   ├── 2025/
│   │   │   ├── 교수학습_및_평가_운영_계획.hwp
│   │   │   └── ...
│   │   └── 2024/
│   │       └── ...
│   └── [학교명]/
│       └── ...
│
└── README.md            # 이 파일
```

## 📄 문서 유형

### 1. 국가 교육과정 문서
- **출처**: 교육부 고시
- **형식**: PDF
- **예시**: 
  - `[별책 8] 수학과 교육과정.pdf`
  - `[별책 9] 과학과 교육과정.pdf`
  - `[별책 10] 실과(기술·가정)/정보과 교육과정.pdf`

### 2. 학교 운영 계획 문서
- **출처**: 개별 학교
- **형식**: HWP, PDF, DOCX
- **예시**:
  - `교수학습_및_평가_운영_계획.hwp`
  - `학년별_교육과정_편성표.hwp`
  - `교과별_평가_계획.hwp`

## 🔧 문서 추가 가이드

### 파일 명명 규칙

#### 국가 교육과정
```
assets/curriculum/[개정년도]/[별책번호]_[과목명]_교육과정.pdf
```
예: `assets/curriculum/2022/[별책 8] 수학과 교육과정.pdf`

#### 학교 운영 계획
```
assets/schools/[학교명]/[년도]/[문서유형].hwp
```
예: `assets/schools/협성고/2025/교수학습_및_평가_운영_계획.hwp`

### 메타데이터 정보

문서 추가 시 다음 정보를 함께 기록:
- **policy_version**: 개정 연도 (예: `2022개정`)
- **scope_type**: `NATIONAL` 또는 `SCHOOL`
- **institution_id**: 학교명 (학교 문서인 경우)
- **document_type**: 문서 유형 (예: `교육과정`, `평가계획`)

## 📊 RAG 시스템 연동

이 폴더의 문서들은 RAG 시스템의 파싱 모듈에 의해 자동으로 처리됩니다:

1. **파싱**: 문서 구조 분석 및 청킹
2. **메타데이터 추출**: 자동 태깅
3. **임베딩**: 벡터 변환
4. **인덱싱**: 벡터 DB 저장

자세한 내용은 `docs/rag/RAG_SYSTEM_PLANNING.md`를 참조하세요.

## ⚠️ 주의사항

1. **저작권**: 교육부 고시 문서는 공공저작물이지만, 학교 문서는 개인정보 보호에 유의
2. **버전 관리**: 동일 문서의 다른 버전은 별도 폴더에 보관
3. **파일 크기**: 대용량 파일은 Git LFS 사용 권장
4. **보안**: 민감한 정보가 포함된 문서는 `.gitignore`에 추가
